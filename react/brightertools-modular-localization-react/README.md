# @brightertools/modular-localization-react

React companion localization administration components for `BrighterTools.ModularLocalization`.

```powershell
npm install @brightertools/modular-localization-react
```

The host app owns localization persistence, sync endpoints, authorization, and translation provider credentials. This package owns reusable admin UI components and typed API adapter contracts.

Build and validate:

```powershell
npm install
npm test
npm run build
npm run pack:dry-run
```
